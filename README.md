# Finance Manager

Консольное приложение для управления личными финансами.  
Позволяет пользователю вести учет доходов и расходов, управлять категориями, устанавливать лимиты и просматривать статистику за выбранный период.

Проект выполнен в учебных целях с использованием Java, Maven и модульной архитектуры.

---

## Функциональность

### Авторизация
- Создание нового аккаунта
- Авторизация по логину и паролю
- Валидация логина и пароля
- Хранение данных пользователя в файле (JSON)

---

### Управление кошельком
- Просмотр текущего баланса
- Просмотр общей суммы доходов
- Просмотр общей суммы расходов

---

### Категории
- Просмотр всех категорий
- Добавление новой категории
- Удаление категории
- Просмотр информации по категории
- Установка лимита на категорию

---

### Операции
- Добавление дохода
- Добавление расхода
- Учет операций по категориям
- Ведение истории операций

---

### Аналитика
- Просмотр доходов за период
- Просмотр расходов за период

Поддерживаемые периоды:
- Последние сутки
- Три дня
- Неделя
- Месяц
- Три месяца
- Год

---

## Как пользоваться приложением

1. Запустите приложение
2. В стартовом меню выберите действие:
    - Создание нового аккаунта
    - Авторизация
3. После успешной авторизации откроется главное меню
4. Выберите нужный пункт меню, вводя номер команды
5. Для выхода из приложения используйте пункт **«Выход»**

---

## Поддерживаемые команды

### Главное меню:

1. Посмотреть все категории
2. Посмотреть остаток
3. Посмотреть общую сумму доходов
4. Посмотреть общую сумму расходов
5. Добавить категорию
6. Удалить категорию
7. Посмотреть информацию по категории
8. Установить лимит на категорию
9. Добавить сумму (доход)
10. Снять сумму (расход)
11. Посмотреть все расходы за период
12. Посмотреть все доходы за период
13. Выход
